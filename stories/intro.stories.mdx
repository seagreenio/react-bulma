# Intro

React components for [Bulma](https://bulma.io/).

Bulma is a popular CSS framework. As its homepage says:

> **Bulma is a free, open source framework that provides ready-to-use frontend components that you can easily combine to build responsive web interfaces.**

The goal of React Bulma is also to do its best to achieve the same as above, which is committed to providing developers with easy-to-use React components.

In order to achieve this goal, the entire component library will be developed around the following two points:

- **Completely convert classes into corresponding component props**
- **TypeScript support**
